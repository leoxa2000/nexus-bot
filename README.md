# NEXUS Trading Bot

Autonom scalping-bot som körs dygnet runt på Railway.

## Vad den gör

- Hämtar riktiga priser från CoinGecko var 15:e sekund
- Skannar 6 kryptovalutor för köp/sälj-signaler
- Handlar med paper trading (låtsaspengar, riktiga priser)
- Sparar all data automatiskt, kommer ihåg allt vid omstart
- Dashboard tillgänglig via webben — öppna från vilken enhet som helst
